# Hardware

## Load device tree
To load the BBBMINI device tree type `startup.sh load`.

## Build dtbo file
To rebuild the dtbo file type `make` and than `make install`to copy the dtbo file to `/lib/firmware`.

### CAN1 UAVCAN
